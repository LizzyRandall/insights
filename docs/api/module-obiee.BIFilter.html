Class: BIFilter

obiee.BIFilter

Object representing a filter. Can be applied to BIQuery objects to generate WHERE clauses in dynamic logical SQL.

Constructor

new BIFilter(column, value, opopt, subjectAreaopt, globalopt)

Parameters:
Name Type Attributes Default Description
column BIColumn

OBIEE column to filter on.
value string

Value to apply to the filter.
op string <optional>
 in

Operator for the filter action. Can be one of a limited set of values: equal, notEqual, notIn, greater, greaterOrEqual, less, lessOrEqual, top, bottom, like, contains, starts, ends, isNull, isNotNull.
subjectArea string <optional>
 BIColumn.subjectArea

Subject area the filter belongs to.
global boolean <optional>
 false

Flag indicating whether the filter is global, i.e. implemented by a drilldown or dashboard prompt.
