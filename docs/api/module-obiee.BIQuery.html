Class: BIQuery

obiee.BIQuery

Object representing a query against the BI server. Uses a subject area andarrays of BIColumn, BIFilter and BISort objects to construct logical SQL to execute against the BI server.

Constructor

new BIQuery(subjectArea, cols, filters, sort)

Parameters:

subjectArea string

OBIEE subject area for the query
cols Array.<BIColumn>

Array of BIColumn objects representing the criteria for the query.
filters Array.<BIFilter>

Array of BIFilter objects used to build the WHERE clause.
sort Array.<BISort>

Array of BISort objects used to build the ORDER BY clause.
