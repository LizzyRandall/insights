displayList(datum, category, measures, event, colourScale, highlight)

Display and position tooltip displaying data in a list format for an XY plot space. Displays one or measure values as a list in the tooltip for comparison.

Parameters:
Name Type Description
datum object

Object with data for all of the measures. Expects measure properties to be arrays of objects with properties name and value.
category string

Property name for the category (dimension) attribute in the datum.
measures Array.<string>

One or more property names indicating the measures in the datum.
event event

Mouse event fired when displaying the tooltip.
colourScale function

D3 scale including the colours that should be tied to the categories.
highlight string

Name of the measure in which to higlight. Will display the number in bold and changes the colour of the border of the tooltip.
