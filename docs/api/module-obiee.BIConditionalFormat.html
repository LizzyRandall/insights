Class: BIConditionalFormat

obiee.BIConditionalFormat

Conditional formatting object.

Constructor

new BIConditionalFormat(sourceID, targetID, val, op, colour, columnMap, dataset)

Parameters:
Name Type Description
sourceID String

Simplified column map ID for the column on which the formatting rule is based.
targetID String

Simplified column map ID for the column on which the formatting style should be applied.
val string | number

Value the rule has to match.
op string

Operator ID for the formatting rule. Can be one of 'equal', 'notEqual', 'greater', 'greaterOrEqual', 'less', 'lessOrEqual' or 'heatmap'.
colour string

Hex colour to apply if the formatting rule is matched.
columnMap Object

Column Map objects have properties containing BIColumn objects or arrays of them.
dataset string

Dataset ID for use with plugins that have multiple datasets allowed. The precise structure is determined by the plugin itself.
