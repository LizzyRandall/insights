Class: BIInteraction

obiee.BIInteraction

Contains information for interactivity between multiple visualisations on a page.

Constructor

new BIInteraction(sourceVis, targetVis, triggeropt, actionopt, columnsopt, handleropt)

Parameters:
Name Type Attributes Default Description
sourceVis BIVisual

Source visualisation object, this contains the trigger for the interaction.
targetVis BIVisual

Target visualisation object, this reacts to the interaction.
trigger string <optional>
 this.getDefaultTrigger

Custom event to be triggered. Possible values are specified by the actions property of the source plugin.
action string <optional>
 this.getDefaultAction

Action to be performed on the target visualisation. Possible values are specified by the reactions property of the target plugin.
columns array <optional>
 getDefaultColumns

Array of column identifiers to pass to the interaction. Possible values defined by the columnMappingParameters property of the source plugin.
handler function <optional>
 insights.generateHandler

JavaScript handler for the UI interaction. This is governed by the framework, but can be optionally passed in to allow deletion or inactivation of interactions.
