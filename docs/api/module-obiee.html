Module: obiee

(static) applyToColumnSets(columnMap, plugin, func)

Applies a funciton to all column maps in a column set. This performs no special action unless the multipleDatasets property of the plugin is set to true. Can be used to build generic objects of the same structure too.

Parameters:
Name Type Description
columnMap object

Column Map objects have properties containing BIColumn objects or arrays of them. The precise structure is determined by the plugin itself.
plugin string

ID of the plugin to which the column map applies. Used to get the multipleDatasets property.
func function

The function to apply to each of the column sets. If multipleDatasets is false, the column set is just a regular column map Object. If multipleDatasets is true, the column set is an object where each key is a different query containing individual column maps.

Callback function to execute on failure.
mask string

Filters on object name, using * as a wildcard.
permMask integer

Integer representing binary interpretation of object permissions for the user. E.g. 3 for read, 65535 for full control.

Retrieves the catalogue permissions for the user and attaches it to a property, permissions.
