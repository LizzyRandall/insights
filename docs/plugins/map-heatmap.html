Map (Heatmap)

Map visualisation using open source plugins LeafletJS and Leaflet Heat. This takes longitude and latitude points and generates a coloured heatmap based on the density of points in the distribution.

Map (Heatmap)

Column Mapping

Name Property Column Type Multiple Required Format Description
Longitude lng Dim Longitude attribute for positioning the points.
Latitude lat Dim Latitude attribute for positioning the points.
Measure measure Fact Measure column to further weight the heatmap algorithm.

Configuration

Name Property Input Type Default Description
Width width Textbox 400 Width of the map in pixels.
Height height Textbox 400 Height of the map in pixels.
Intensity maxZoom Textbox 8 Intensity of the heat map colours with lower numbers for higher intensity. This is actually related to the zoom level of the map plugin, such that the number chosen is the zoom level at which the maximum colour will always be displayed.
Colour Scale colours Palette Cool-Scale Colour palette for choosing the heatmap gradient.
Map Tiles mapTile Maptile Default Defines the images used for the map background in the plugin.
