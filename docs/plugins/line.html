Line Chart

  • Column Mapping
  • Configuration
  • Actions
  • Reactions

Line chart visualisation for continuous data, can be used as an area chart by setting the appropriate configuration setting. Secondary Y axes can be used when there are two measures to scale them appropriately regarding their values. When a date column is used as the category property the dataset will be modified to be continuous. This means that if OBIEE is missing records for certain days, they will be assigned a value of 0.

Line Chart

Column Mapping

Name Property Column Type Multiple Required Format Description
Category category Dim Single column to be used as the x-axis of the line chart. Should contain continuous values.
Measure measure Fact Aggregatable measure column to be used for the line in the y-axis. Multiple measures can be selected, which will render multiple lines for each measure value in different colours. Cannot have multiple measures and a Vary By Colour column.
Vary By Colour vary Dim Single column to pivot a single measure into multiple lines of different colours. Will not work when using multiple measures.
Hidden hidden Hidden Hidden columns that can be used for conditional formatting without being displayed. It is important that adding this column does not alter the granularity of the query, i.e. the hidden column should be of the same level, or less descriptive than the category column.

Configuration

Name Property Input Type Default Description
Width width Textbox 400 Width of the chart in pixels.
Height height Textbox 300 Height of the chart in pixels.
Point Size pointSize Textbox 2 Radius of the points to display in pixels.
Line Width strokeWidth Textbox 2 Stroke width of the line.
Vertical horizontal Checkbox Displays with horizontal orientation as opposed to vertical.
Show Legend legend Checkbox true Boolean which if true will show the legend.
Second Axis secondAxis Checkbox Boolean which if true will display a second axis on the right hand side of the chart. This will only work if specifically two measures have been defined, and is not compatible with vary by colour.
Drag Select dragSelect Checkbox true Boolean which if true allows the user to drag on the main chart area to zoom into that are of the chart.
Mini-Chart Navigator brushNav Checkbox true Boolean which if true will render a miniature chart underneath the original that can be used for navigation. The user can drag on the miniature chart to scroll and zoom the main chart.
Shade Area shadeArea Checkbox Boolean which if true shades the area under the line.
Colours colours Palette Flat-UI Colour palette to use for discrete series colours
