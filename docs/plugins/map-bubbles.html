Map (Bubbles)

Map visualisation using open source plugin LeafletJS. This shows bubbles based on points positioned by longitude and latitude and sized based on a measure value. Hovering over the bubbles will display a tooltip. Clicking the pencil icon at the top of the map will allow the user to select multiple points by drawing a shape freehand.

Map (Bubbles)

Column Mapping

Name Property Column Type Multiple Required Format Description
Description desc Dim Attribute column giving the descriptive name for a point.
Longitude lng Dim Longitude attribute for positioning the points.
Latitude lat Dim Latitude attribute for positioning the points.
Measure measure Fact Measure attribute scaling the size of the points. Choosing multiple measure will allow the legend to act as a measure selector.
Vary By Colour vary Dim Single column to pivot a single measure into multiple series of different colours. Will not work when using multiple measures.
