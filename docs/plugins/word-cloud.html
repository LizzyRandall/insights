Word Cloud

  • Column Mapping

Word cloud (also known as tag cloud or weighted) list visualisation. Here the attribute word will be sized based on the equivalent measure value. The positioning on the canvas is determined by one of two configurable algorithms. The word colours are also configurable.

Word Cloud

Column Mapping

Name Property Column Type Multiple Required Format Description
Word word Dim Attribute column representing the words for the cloud.
Frequency freq Fact Measure column giving the data for the size weightings.

Configuration

Name Property Input Type Default Description

Height height Textbox 400 Height of the canvas in pixels.

Maximum Font Size maxSize Textbox 36 Maximum font size for the word tags.
Font font Font Open Sans Font of the word tags.
Number of Orientations orientations Textbox 5 Choose a number of different orientations between the given angles (min -90, max 90).
From Angle fromAngle Range -60 Minimum angle of orientation for a word in the cloud.
To Angle toAngle Range 60 Maximum angle of orientation for a word in the cloud.
Colours colours Palette Flat-UI Colour pelette defining possible colours for the words.
