Pivot table created using a modified version of Nicolas Kruchten's JS plugin. The pivot table has a fixed height and width and the content is scrollable both horizontally and vertical when necessary.

Pivot Table

Column Mapping

Name Property Column Type Multiple Required Format Description
Rows rows Any Attributes to be shown as rows in the pivot table. Can select multiple.
Columns columns Any Attributes to be shown as columns in the pivot table. Can select multiple.
Measures measures Fact Measures to display in the cells. Must be numeric measures that can be aggregated. Can choose multiple.
Hidden Measures hidden Hidden Measures that are in the dataset but are not displayed in the table. This can be used for conditionally formatting rows based on certain columns.

measures Config Options

Name Property Input Type Default Description
Default Aggregator aggregator Dropdown Sum Aggregate function to use for the measure. E.g. Sum, Mean, Median.
Hide Values hide Checkbox Hide the values of this measure from the table, but display totals.

Configuration

Name Property Input Type Default Description
Width width Textbox 300 Width for the scrollable content of the table.
Height height Textbox 300 Height for the scrollable content of the table.
Column Width columnWidth Textbox 100 Fixed width of each column in the pivot table.
Allow Controls controls Checkbox true Allow the user to access the pivot control panel to move/exclude columns.
Values as Columns valuesAsCols Checkbox true Boolean to choose to display value headings as columns or rows.
Type renderer Radio Table Choose the rendering mode, table and table barchart available.
Theme Colour themeColour Colour #5DA5DA Colour scheme for the table.

Conditional Formatting

Name ID Require Value Description
Heatmap heatmap Colours cells in table based on value. As such a value for the formatting rule is not required, instead the root colour should be chosen.
