The Title

Scatter Chart

  • Column Mapping
  • Configuration
  • Actions
  • Reactions

Scatter chart visualisation, displaying points on a plot space of two measures against one another. Can be used as a bubble chart as well by setting a third measure to be the size variable. Also can calculate a linear regression and R-squared value indicated correlation.

Scatter Chart

Column Mapping

Name Property Column Type Multiple Required Format Description
Measure (X) measureX Fact Measure to select on the X axis. Currently does not support attribute information here.
Measure (Y) measureY Fact Measure to display on the Y axis.
Group By group Dim Attribute to group the measures by. Will determine the number of points plotted. If no attribute was selected here, OBIEE would aggregate the two measures into a single point.
Vary By Colour varyColour Dim Will vary the colour of the points based on the attribute selected.
Vary By Size varySize Fact Will vary the size of hte points based on the measure selected. This will effectively turn the chart into a bubble chart.

Configuration

Name Property Input Type Default Description
Width width Textbox 400 Width of the chart in pixels.
Height height Textbox 400 Height of the chart in pixels.
Minimum Point Size minPointSize Textbox 2 Minimum radius for the points.
Maximum Point Size maxPointSize Textbox 6 Maximum radius for the points. Only relevant if Vary By Size has been populated.
Drag to Select selectBox Checkbox true Allows functionality to drag plot area, selecting points in the area and aggregating the values of each selected point.
Linear Regression lr Checkbox true Calculates linear regression for the chart, displaying a dashed line for the correlation and the R-squared value when hovering over the line.
X Axis Title xTitle Textbox default Text input for the title of the X-axis.
Y Axis Title yTitle Textbox default Text input for the title of the Y-axis.
Colours colours Palette Flat-UI Colour palette for the series of points, colours past the first are only relevant if Vary By Colour is populated.
