The Title

Line/Bar Chart

  • Column Mapping
  • Configuration
  • Actions
  • Reactions

Displays a bar and line chart on the same plot, with two Y axes but the same X axis. Should be used for displaying discrete information. Bars can be displayed as stacked as well as making the whole chart horizontal by editing the configuration properties.

Line/Bar Chart

Column Mapping

Name Property Column Type Multiple Required Format Description
Category category Dim Single column to be used as the x-axis of the bar chart. Should contain discrete values.
Bar Measure barMeasure Fact Aggregatable measure column to be used for the line in the y-axis. Multiple measures can be selected, which will render multiple bars for each measure value in different colours.
Line Measure lineMeasure Fact Aggregatable measure column to be used for the line in the y-axis. Multiple measures can be selected, which will render multiple lines for each measure value in different colours.
Hidden hidden Hidden Hidden columns that can be used for conditional formatting without being displayed. It is important that adding this column does not alter the granularity of the query, i.e. the hidden column should be of the same level, or less descriptive than the category column.

Configuration

Name Property Input Type Default Description
Width width Textbox 400 Width of the chart in pixels.
Height height Textbox 300 Height of the chart in pixels.
Point Size pointSize Textbox 3 Radius of the points to display in pixels.
Line Width strokeWidth Textbox 2 Stroke width of the line.
X Axis Title xTitle Textbox default Text input for the title of the x-axis.
Y Axis Title yTitle Textbox default Text input for the title of the left y-axis.
Second Y Axis Title y2Title Textbox default Text input for the title of the right y-axis.
Mini-Chart Navigator brushNav Checkbox true Boolean which if true will render a miniature chart underneath the original that can be used for navigation. The user can drag on the miniature chart to scroll and zoom the main chart.
Show Legend legend Checkbox true Boolean which if true will show the legend.
Stacked Bar stacked Checkbox Display multiple bars as stacked instead of inline.
Horizontal horizontal Checkbox Display chart horizontal with orientation rather than vertical.
Default Sort Direction sortDirDefault Radio Descending Sort direction upon loading the chart.
Default Sort Column sortColDefault Textbox 1 he default column to sort by. 1 is the first measure, 0 refers to the category dimension. Increasing the number will choose the respective measure in the series.
Sort Control sortControl Checkbox true Indicates whether to show the sort control bar or not.
Colours colours Palette Flat-UI Colour palette to use for discrete series colours.
