Map visualisation using open source plugin LeafletJS. This shows regions coloured as a heatmap, darker colours for higher values and vice versa. Hovering over the regions will display a tooltip. Works by using a topoJSON file uploaded to the deployment on the server (in the topojson folder). This file will have the description of the regions as well as an attribute that can be used to tie the OBIEE dataset to the map.

Map (Tiles)

Column Mapping

Name Property Column Type Multiple Required Format Description
Code code Dim Attribute representing the code that matches the attribute in the topoJSON file.
Description desc Dim Descriptive field for a given region.
Measure measure Fact Measure values used to colour the regions. Darker colours represent higher values and vice versa. Choosing multiple measures will cause the legend to become column selector.
Vary By Colour vary Dim Single column to pivot a single measure into multiple series of different colours. Will not work when using multiple measures.

Configuration

Name Property Input Type Default Description
Width width Textbox 400 Width of the map in pixels.
Height height Textbox 400 Height of the map in pixels.
TopoJSON File topojson Map TopoJSON file existing in the topojson directory.
Feature Code featureCode Mapcode TopoJSON attribute linking the OBIEE dataset to the map.
Feature Description featureDesc Mapcode TopoJSON attribute with the descriptive name for each feature.
Colour Scale Type scaleType Radio Linear Scale for the colouration, choosing between linear and quantile. Quantile is better for displaying data that has a very high range.
Opacity opacity Range 0.75 Opacity of each feature on the map.
Show Legend legend Checkbox true Optionally show or hide the legend.
Style Type styleType Radio Series Picker Defines whether the legend can be used as a colour picker for multiple measures, or whether colours will be determined on the most prominent value.
Series Scale seriesScale Radio Individual Defines whether each data series has its own scale, or if the is a single common scale across all series.
Null Colour nullColour Colour #CCCCCC Colour for regions which do not map to any rows in the OBIEE dataset.
Colours colours Palette Flat-UI Colour palette to use for discrete series colours. Each colour will be converted to a gradient when displaying each measure.
Map Tiles mapTile Maptile Default Defines the images used for the map background in the plugin.
