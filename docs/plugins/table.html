Basic table visualisation, created using the AG Grid plugin to create a scrollable table of results. Features include sorting by column and searching by text. Neither of these require going back to the database. Users can also change the order and width of columns as well as freezing the panes by pinning certain columns to the left or right.

Table

Column Mapping

Name Property Column Type Multiple Required Format Description
Column columns Any Choose multiple columns to add to the table. Will display left to right in the order selected here.
Hidden hidden Hidden Can use hidden columns that will not display in the table but will affect the data set. Can be used to conditionally format data based on undisplayed information.

columns Config Options

Name Property Input Type Default Description
Column Width width Textbox -1 Width of the column in pixels. Set to -1 for auto sizing.
Pinned pinned Radio None Choose to pin the column to the left or right of the table so it is always visible when scrolling.

Configuration

Name Property Input Type Default Description
Width width Textbox 400 Width of the table in pixels.
Height height Textbox 400 Height of the table in pixels.
Theme Colour themeColour Colour #5DA5DA Sets the colour for the table. Will have a solid line below the header and a lighter shade of the colour for a hover over of each row.
Font font Font Open Sans String input for the font to display the table in.
Font Size fontSize Textbox 11 Size of the font to display in the table.
Wrap Headers wrapHeader Checkbox Wraps header text and automatically makes header taller to compensate.
